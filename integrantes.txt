Instrucciones para ejecutar las pruebas:
========================================

1. Clonar o descargar el repositorio:
   git clone https://github.com/santiagoguti17/DYAS-TDD-SantiagoGutierrez.git

2. Navegar al directorio del proyecto:
   cd DYAS-TDD-SantiagoGutierrez

3. Compilar el proyecto:
   mvn clean compile

4. Ejecutar las pruebas unitarias:
   mvn clean test

5. Generar reporte de cobertura JaCoCo:
   mvn jacoco:report

6. Abrir el reporte de cobertura:
   Abrir archivo: target/site/jacoco/index.html en un navegador web
